Currency Convert API

To run the server please use:

DEBUG=myapp:* npm start

Ensure you are in the currencyConvert folder.

The server shall run on localhost:8008

Actions

Post /pounds

- this converts into pounds

- Instructions are added the stack whic is stored on the server.

Post/euros

- this converts into euros

- Instructions are added the stack whic is stored on the server.

Get/stack

Returns the stack. Can be used for the history.

Get/pop

Removes the last added instruction

Get/reset

Removes the last added instruction

Get/

Returns the homepage with all the functionalty on it
