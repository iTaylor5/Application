# Application

## currencyConvertServer

Navigate to the currencyConvertServer folder in terminal.

    Run the below command:
    
        DEBUG=currencyConvertServer:* npm start

    Open in browser:
    
        Open http://localhost:8008/ in the browser

    "/" Opens the index.html page
